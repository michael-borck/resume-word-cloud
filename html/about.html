about page

This is a sample about page that demonstrates how the overlay system works with separate HTML files.

sample content

Each overlay page is a separate HTML file that contains only the content div. The main index.html page loads this content dynamically when the user clicks on a word cloud item.

You can add as many sections as you want, with rich formatting including:

  • Lists (ordered and unordered)
  • Headings using title-block divs
  • Paragraphs with proper spacing
  • Links to external resources
how it works

The JavaScript fetches this HTML file, extracts the #content div, and inserts it into the overlay. This approach keeps your content modular and easier to maintain than having everything in a single file.
